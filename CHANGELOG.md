# Changelog

## v1.0.0 - 2026-06-13

### Added

- Landing page da Bússola FCTE.
- Fluxo de diagnóstico interativo para calouros.
- Algoritmo de afinidade baseado em distância e função exponencial.
- Páginas de exploração dos cursos com disciplinas, mitos, projetos e mini missões.
- Estrutura estática pronta para GitHub Pages, Vercel, Netlify ou Cloudflare Pages.
- Documentação de arquitetura, contribuição e deploy.
- Validador local para checar links e arquivos estáticos.

### Changed

- Código organizado em `assets/`, `src/css`, `src/js`, `src/data`, `docs` e `tools`.
- Removidas dependências e arquivos gerados que não eram necessários para produção.

### Notes

- O MVP não usa login, backend ou banco de dados.
- As respostas ficam apenas no navegador durante a sessão.
- Dados de mercado são orientativos e devem ser revisados antes de uso institucional.
